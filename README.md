# QuieroVerApp

QuieroVerApp es una aplicación personal para iOS pensada para guardar películas y series que quiero ver, consultar información básica sobre ellas y archivarlas como vistas con una nota personal.

La app utiliza metadatos de TMDb para obtener información como posters, sinopsis, reparto, dirección/creación, año y géneros.

## Objetivo

El objetivo del proyecto es crear una biblioteca audiovisual privada, sencilla y visual, centrada en dos estados principales:

* **Quiero ver**: películas y series pendientes.
* **Vistas**: películas y series ya vistas, con valoración personal y nota libre.

## Estado del proyecto

Proyecto en desarrollo.

## Uso

Aplicación local-first para uso personal y no comercial.

## Nota

Este proyecto utiliza la API de TMDb para obtener metadatos de películas y series. La API key no está incluida en el repositorio y debe configurarse localmente.
